# Contributing to Yashigani

External contributions are welcome to **this repository (`yashigani`)** via pull request.

- Fork the repository and open a pull request against `main`.
- Every change requires **review and approval by a maintainer** (see `.github/CODEOWNERS`) before it can be merged — there are **no direct commits to `main`**, and no bypass.
- If you push new commits after an approval, the pull request must be **re-approved**.
- CI on pull requests from forks runs **only after a maintainer approves** the run.
- Keep pull requests focused; describe the change and the rationale.

Our other public repositories are **internal-only** and do not accept external contributions.

By contributing, you agree your contribution is licensed under this repository's `LICENSE`.
